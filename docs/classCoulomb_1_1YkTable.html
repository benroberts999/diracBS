Detailed Description

Calculates + stores Hartree Y functions + Angular (w/ look-up)

Also stores an Angular::Angular::Ck_ab table (publically).

Note: Look-up must be done such that: Fa MUST be member of a_orbitals, Fb of b_orbitals. Cheking is done with assers, so only in 'dev' or 'debug' mode - ranges not checked in release mode.
