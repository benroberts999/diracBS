Takes existing radial vector, multiplies by:

a(Z) * a0 * exp( - b * r / a0). a0 = alpha = 1/137. b=1 by default. A should be fitted. a(Z) = 1.0 + 28.5/Z nb: can give it an empty vector, to just get the exponential function
