bool IO::UserInput::check ( const std::string &  in_block,
const std::vector< std::string > &  options 
) const

Checker. Returns false if any of the input options in the 'in_block' block are not listed in the given 'options' list.

This checks to ensure we aren't trying to set an option that doesn't exist, or if we've spelled an option wrong. Otherwise, these would simply be ignored and default values used in the calculations, which is likely an error (and an easy one to miss)
